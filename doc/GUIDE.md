# Guide

## Main Library and Pattern
- React js
- Hooks
- Typescript
- ESLINT
- Jest
- S3 Web hosting

## Quick start
1. Run <b>"npm install"</b>  in order to install dependencies.
2. At this point you can run <b>"npm start"</b> or <b>"npm run start"</b> to see the example app at http://localhost:3000.
